# Software and Automation for Data Science

Welcome to the course materials for DS5111 at the University of Virginia. This book serves as a living document for our exploration into building robust, automated pipelines and modern software engineering practices within the data science lifecycle.

## Course Objectives
* **Reproducibility:** Mastering environments and dependency management.
* **Automation:** Leveraging Makefiles, GitHub Actions, and agentic workflows.
* **Scale:** Moving from local scripts to cloud-integrated pipelines.
